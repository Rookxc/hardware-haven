Voice Commands

Command in () is optional

Command: Hello
Action: Says "Hello"
Command: Hello My name is *name
Action: Greets with the provided name
Command:(Please) Help
Action: Shows voice command options
Command: (Please) search *name
Action: Searches for the provided term
Command: (Please) clear search
Action: Clears the search input
Command: (Please) click *name
Action: Clicks the button with the specified name
Command: (Please) set *value to (item) *input
Action: Sets the value of the specified input
Command: (Please) add *value to (item) *input
Action: Adds value to the specified input
Command: (Please) subtract *value from (item) price
Action: Subtracts value from the input price
